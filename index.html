Web3 Wallet Inspector

Look up any Ethereum-compatible wallet address — balance, transaction count, and network info. No wallet connection or API key required.

Built with ethers.js and free public RPC endpoints (no API key needed). Data is read directly from the blockchain client-side — this page never sends your lookups to any third-party server. Useful as a quick reference tool when testing dApps or verifying testnet/mainnet transactions during Web3 QA work.
