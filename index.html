🔍 Web3 Wallet Inspector

Look up any Ethereum-compatible wallet: balance, transaction count, account type, and block data. Client-side, no API key or wallet connection needed.

✓ Balance ETH/MATIC held
✓ Nonce Transaction count
✓ Type Wallet or contract
✓ Block Current chain state
Built with ethers.js v6 and free public RPC endpoints from PublicNode (no signup required).

Privacy-first: All data is read directly from the blockchain client-side. Your searches are never sent to any third-party server.

Use case: Quick reference tool for Web3 QA testing, dApp verification, or validating testnet/mainnet transactions.

Rate limiting? If you hit public RPC limits, swap in your own free tier key from Alchemy or Infura by editing RPC_ENDPOINTS in app.js.
